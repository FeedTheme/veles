Neural networks done right.

veles.znicz NN engine focuses on performance and hackability. It has little hard-coded entities and enables training all the widely recognized topologies, such as fully connected nets, convolutional nets, reccurent nets.

In-depth introspection and rich debugging facilities help developing nets fast and predictable.

Contributions are welcome.

Send pull requests on GitHub which get automatically converted into Gerrit patchsets. Watch nightly tests run on Jenkins.

Veles is written in Python, OpenCL and CUDA (Jinja2 templated), web parts are built with Gulp and use SASS, ES6 and Node.js.

There is plenty of work to do: port to Windows and Mac OS X, add CuDNN and other engines, add more reporting options, improve the distributed operation, etc. etc.
